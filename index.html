Question 1:

What are the differences between Web1, Web2 and Web3?

Answer:


Web1 is a "static" web that has been used in the 90's, When it was mostly used for information and text with not much images or videos.

Web 2 is a "Social" web that has started in the 2000's, Web2 included blogs, video sharing social platforms (facebook, twitter etc..),
Users has greater control and communication over their online usage.

Web3 is the latest evolution of the web that has started in 2020,
and still in it's beginning and it aims to created greater control over data and to be more secured & private then Web2.

Answer:
